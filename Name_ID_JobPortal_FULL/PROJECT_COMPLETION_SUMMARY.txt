### Skill Matching Algorithm
```python
# 1. Get jobseeker's skills
skills = set(jobseeker.skills.lower().split(','))

# 2. For each job, get required skills
job_skills = set(job.skills.lower().split(','))

# 3. Find intersection (matching skills)
matched = skills & job_skills

# 4. If intersection exists, job is a match
if matched:
    display_job_on_dashboard()
```
